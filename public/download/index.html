Dataset Contents#

What’s Included#

  • Training Set: Annotated samples for model training
  • Validation Set: Samples for model validation and hyperparameter tuning
  • Test Set: Samples for final model evaluation
  • Annotations: Expression labels and occlusion type information
  • Metadata: Subject demographics, capture conditions, and technical specifications
  • Baseline Results: Performance benchmarks from standard methods
  • Documentation: Detailed dataset description and usage guidelines
